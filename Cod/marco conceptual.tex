LoRaWAN es una especificación diseñada específicamente para dispositivos de bajo consumo de alimentación, que operan en redes de alcance local, regional, nacionales o globales.\\

Ubuntu MATE es una distribución Linux\\

Ubidots es una plataforma de IoT que permite enviar datos de sensores a la nube, configurar tableros y alertas, conectarse con otras plataformas, usar herramientas de analítica y arrojar mapas de datos en tiempo real.\\

Mosquitto es un software de código abierto y gratuito, bajo licencia EPL/EDL, y que actúa como un broker o intermediario de mensajes a través del protocolo MQTT.\\

Raspbian es un sistema operativo libre, basado en Debian (Linux), y presenta una amplia comunidad de desarrolladores.

Contiki OS es un sistema operativo de código abierto desarrollado para uso en un número de pequeños sistemas pasando desde ordenadores de 8-bit a sistemas embebidos sobre microcontroladores, incluyendo nodos de redes de sensores.\\

NODE-RED es una herramienta de programación basada en JavaScript, este se programa utilizando flujos que se ejecutan de forma paralela, usando como interfaz de desarrollo un navegador web.\\

NodeMCU es una plataforma IoT de código abierto. Incluye el firmware que se ejecuta en el SoC Wi-Fi ESP8266 de Espressif Systems y el hardware que se basa en el módulo ESP-12.\\

Thinger.io es una plataforma IoT Open Source que ofrece herramientas con las que cualquier empresa tradicional puede integrar sus productos o infraestructuras con las tecnologías 4.0 de forma sencilla, económica y escalable, aunque no cuenten con un equipo especializado en IoT.\\

El diseño web adaptable (responsive), es una filosofía de diseño y desarrollo cuyo objetivo es adaptar la apariencia de las páginas web al dispositivo que se esté utilizando para visitarlas.\\

RIOT OS es un sistema operativo pequeño para networked, sobre sistemas con memoria limitada con un enfoque específico sobre dispositivos de Internet de las Cosas.\\

Angular es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.\\

Firebase es una plataforma para el desarrollo de aplicaciones web y aplicaciones móviles lanzada en 2011 y adquirida por Google en 2014.\\

Heroku es una solución de Plataforma como Servicio (PaaS) basada en la nube para que el cliente solo se preocupe de desarrollar su aplicación mientras Heroku se encarga de la infraestructura que hay detrás.\\

API RESTful es una interfaz que dos sistemas de computación utilizan para intercambiar información de manera segura a través de Internet.\\

IBM Watson IoT Platform es un servicio totalmente gestionado y alojado en la nube que facilita la derivación de valor de los dispositivos IoT
