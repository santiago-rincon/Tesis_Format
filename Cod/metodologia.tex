\section{\fontsize{20}{0}{\selectfont{Metodología}}}
\rule{\textwidth}{1pt}\\

En esta sección se describen las herramientas, estrategias, técnicas y procedimientos requeridos para satisfacer el objetivo principal del proyecto y llegar a una solución de la problemática. El desarrollo de esta se asume tratando individualmente cada objetivo específico detallando las actividades necesarias para su total cumplimiento. Las fases para el desarrollo del poyecto se observan en la Figura \ref{meto}.
\begin{figure}[H]
	\centering
	\shadowbox{\includegraphics[width=0.88\textwidth]{meto}}
	\caption{Fases para el desarrollo del proyecto}
	\label{meto}
\end{figure}
En la primera fase se definirá la plataforma en la cual se diseñará la interfaz gráfica ya sea escritorio, web, Android, IOS u otra. Posteriormente, se realizará la revisión de los diferentes lenguajes de programación que pueden ser empleados para la elaboración de la interfaz gráfica de acuerdo con el entorno anteriormente establecido, para finalmente profundizar en los diferentes conceptos y herramientas disponibles del lenguaje en cuestión para la elaboración de la primera versión de la interfaz.\\

Una vez diseñada la parte gráfica (estética de la aplicación) en la segunda fase se diseñará un modelo en 3D de la estación de supervisión con el fin de incluirla dentro de la aplicación para posteriormente realizar la vinculación del nodo central de la red y la interfaz gráfica con un servidor común en la nube para la lectura de datos y manipulación de los diferentes sensores la red inalámbrica WSN (Wireless Sensor Networks) a través de internet.\\

Finalmente, en la última fase se plantearán diversas pruebas de lectura y manipulación de los diferentes nodos de la red en un entorno local, con el fin de identificar los fallos (bugs) presentados, esto con el objetivo de realizar la oportuna corrección de hardware o software (según sea el caso) para así, obtener un correcto funcionamiento de la red. Con las pruebas locales superadas se procederá a realizar una prueba final de toda la red de sensores de manera remota para verificar su correcto funcionamiento.\\

Al terminar cada una de las fases anteriormente mencionadas se realizará la creación de un manual de usuario y cartilla de implementación para todo el sistema completo, esto con el objetivo de capacitar a la comunidad externa y académica interesada. Además, se desarrollará un documento para participación en eventos científicos.
